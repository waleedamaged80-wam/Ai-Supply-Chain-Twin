## 🎯 Overview

AI Supply Chain Twin is a production-ready web application for supply chain professionals to optimize inventory policies, test disruption scenarios, and make data-driven decisions using Monte Carlo simulation and AI-powered insights.

**🌐 Try the live demo:** [https://ai-supply-chain-twin.streamlit.app](https://ai-supply-chain-twin.streamlit.app)

---

## ✨ Features

### Core Capabilities
- 🏭 **Multi-Echelon Network Simulation** - Factory → Regional DC → Local Store
- 📊 **Monte Carlo Analysis** - 50+ iterations for robust statistical insights
- 🤖 **AI-Powered Decision Engine** - Intelligent recommendations and risk scoring
- 📈 **Real-Time Visualization** - Interactive dashboards and performance analytics
- 🎯 **Scenario Planning** - Natural language disruption testing

### Advanced Features
- **Configurable Economic Parameters** - Unit costs, margins, lead times
- **Risk Assessment** - Multi-dimensional scoring (0-100 scale)
- **Performance Metrics** - Service level, inventory turns, profit analysis
- **Portfolio Management** - Analyze hundreds of SKUs simultaneously
- **Cost Optimization** - Holding, stockout, and transport cost analysis

## 🚀 Quick Start

### Option 1: Use Online (No Installation)
**Click here:** [Live Demo](https://ai-supply-chain-twin.streamlit.app)

### Option 2: Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/ai-supply-chain-twin.git
cd ai-supply-chain-twin

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run ai_twin_ultimate.py
```

## 📊 How It Works

**1. Upload Data** → CSV with SKU and demand information  
**2. Configure** → Set costs, lead times, policies  
**3. Plan Scenarios** → "Demand surge 25%", "Factory delay 10 days"  
**4. Analyze** → Get insights, risk scores, recommendations

```
🏭 Factory (100,000 units) → 🏢 Regional DC → 🏪 Local Store → 👥 Customers
```
